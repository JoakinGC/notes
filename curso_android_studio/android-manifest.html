¿Qué es Android Manifest?

El AndroidManifest.xml es un archivo esencial en cualquier aplicación Android. Se encuentra en el directorio raíz de cada proyecto y su función principal es proporcionar información fundamental sobre la aplicación al sistema operativo.

Funciones principales de Android Manifest:

  1. Declaración de componentes de la aplicación

    • Define las actividades (<activity>), servicios (<service>), receptores de difusión (<receiver>) y proveedores de contenido (<provider>) que forman parte de la aplicación.

  2. Permisos requeridos

    • Se especifican los permisos que la aplicación necesita para acceder a funciones del sistema o hardware, como acceso a internet, ubicación, cámara, almacenamiento, etc.
    <uses-permission android:name="android.permission.INTERNET"/>
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>

  3. Configuración del launcher y actividades

    • Se define cuál es la actividad principal (la que se abre al iniciar la aplicación).
    <activity android:name=".MainActivity">
    <intent-filter>
        <action android:name="android.intent.action.MAIN"/>
        <category android:name="android.intent.category.LAUNCHER"/>
    </intent-filter>
</activity>

  4. Versión y compatibilidad de la aplicación

    • Se establece la versión de la aplicación y la versión mínima y máxima de Android en la que puede ejecutarse.
    <uses-sdk
    android:minSdkVersion="21"
    android:targetSdkVersion="33"/>

  5. Declaración de temas y configuraciones globales

    • Define estilos, configuraciones de orientación de pantalla, modo nocturno, entre otros.
    <application
    android:theme="@style/Theme.MyApp"
    android:allowBackup="true"
    android:supportsRtl="true">

Ubicación del archivo

El archivo AndroidManifest.xml se encuentra en:
app/src/main/AndroidManifest.xml

Ejemplo de un AndroidManifest.xml básico

<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.myapp">

    <!-- Permisos requeridos -->
    <uses-permission android:name="android.permission.INTERNET"/>
    
    <!-- Configuración de compatibilidad -->
    <uses-sdk android:minSdkVersion="21" android:targetSdkVersion="33"/>

    <application
        android:allowBackup="true"
        android:theme="@style/Theme.MyApp">
        
        <!-- Actividad principal -->
        <activity android:name=".MainActivity">
            <intent-filter>
                <action android:name="android.intent.action.MAIN"/>
                <category android:name="android.intent.category.LAUNCHER"/>
            </intent-filter>
        </activity>

    </application>
</manifest>

Importancia de Android Manifest

Sin este archivo, Android no sabría cómo ejecutar la aplicación, qué componentes tiene o qué permisos necesita. Es esencial para el correcto funcionamiento y distribución de cualquier aplicación Android.

Siempre revisa y configura correctamente este archivo para evitar errores al ejecutar la aplicación.
