Bootstrap Grid y Flexbox

Clases de la Grid en Bootstrap

Las clases utilizadas en la grid son las siguientes:

  • .row
  • .col
  • .col-sm-
  • .col-md-
  • .col-lg-
  • .col-xl-

Row

Define una fila en la grid.

Col

Define una columna dentro de una fila.

Las demás variantes de .col permiten adaptar el diseño a diferentes tamaños de pantalla.

Tamaños de pantalla (Breakpoints)

Los breakpoints son puntos de quiebre en los cuales se aplican diferentes estilos según el tamaño de la pantalla:

  • xs: Extra pequeño
  • sm: Pequeño
  • md: Mediano
  • lg: Grande
  • xl: Extra grande
  • xxl: Muy grande

Estos breakpoints permiten cambiar los estilos de los elementos según el tamaño de la pantalla, también conocido como viewport.

Breakpoints

Cada breakpoint define una dimensión en la que podemos cambiar el estilo, tamaño o alineación de los elementos.

Cuando se usa .col sin especificar un tamaño, por defecto se aplica a xs.

Para especificar un tamaño específico, utilizamos sm, md, lg, etc.:

<div class="col-sm">...</div>
<div class="col-md">...</div>

Container

Un contenedor en Bootstrap está compuesto por 12 columnas y agrupa las filas dentro de la grid.

Tipos de contenedores

  • .container → Crea un ancho fijo que se adapta al tamaño de la pantalla.
  • .container-fluid → Ocupa el 100% del ancho disponible.

Contenedores específicos por tamaño de pantalla

  • .container-sm
  • .container-md
  • .container-lg
  • .container-xl
  • .container-xxl

Estos funcionan igual que las columnas (col-sm, col-md…), permitiendo definir un ancho que cambia en su punto de quiebre.

Para centrar texto dentro de un contenedor, usamos .text-center.

Flexbox en Bootstrap

Bootstrap utiliza Flexbox para distribuir y alinear elementos de forma flexible.

Clase principal de Flexbox

  • .d-flex → Habilita el uso de Flexbox en un contenedor.

Propiedades de Flexbox

1. flex-direction

Define la dirección principal de los elementos dentro del contenedor.

Valores posibles:

  • row
  • row-reverse
  • column
  • column-reverse

2. justify-content

Controla la distribución de los elementos en el eje principal.

Valores:

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

Clases equivalentes en Bootstrap:

  • .justify-content-start
  • .justify-content-end
  • .justify-content-center
  • .justify-content-between
  • .justify-content-around
  • .justify-content-evenly

3. align-items

Define la alineación de los elementos en el eje secundario.

Valores:

  • flex-start
  • flex-end
  • center
  • baseline
  • stretch (los elementos ocupan todo el eje secundario)

Clases en Bootstrap:

  • .align-items-start
  • .align-items-end
  • .align-items-center
  • .align-items-baseline
  • .align-items-stretch

4. flex-wrap

Determina si los elementos dentro de un contenedor deben ajustarse a nuevas filas o columnas.

Valores:

  • nowrap (por defecto)
  • wrap
  • wrap-reverse

Clases en Bootstrap:

  • .flex-nowrap (por defecto)
  • .flex-wrap
  • .flex-wrap-reverse
