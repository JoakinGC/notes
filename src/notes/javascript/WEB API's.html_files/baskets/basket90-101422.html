.alert(), confirm() y  prompt()

Son métodos dentro de window.

Los nuevos métodos al igual que los anteriores tambien no es necesario colocar window.

open("") dentro de las comillas va la direccion url que queramos redirigir al usuario pero no es buena practica utilizar pestañas emergentes ya que afecta al SEO

close() cierra la pestaña que estamos actualmente

si guardamos la direcion con window.open() en una variable si la combinamos con close se cierra la pestaña que se abrio

$btnAbrir.addEventListener("click",(e) =>{

   //window.open("https://jonmircha.com";);

   ventana = open("https://jonmircha.com";);

})


$btnCerrar.addEventListener("click",(e) =>{

   //window.close();

   //close();

   ventana.close();

})
