Desde HTML 5 se creo los Data-Attributes.

charset es un atibuto de html

Data Atributes brinda informacion a los enlaces.

Existen dos maneras de trabajar con nuestros valores y tambien brindar valores.

Una forma es colocando el siguiente codigo para acceder al atributo:

console.log(document.documentElement.lang);

Otra forma es con:

console.log(document.documentElement.getAttribute("lang"));

Aunque aparentemente se vean iguales la diferencia erradica en que .documentElement.getAttribute() trae solo el atributo de la etiqueta y no toda la etiqueta como si usaramos .querySelector() por eso cuando se usa combinado getAttribute no traera toda la etiqueta sino solo lo que pedimos.

console.log(document.querySelector(".link-dom").href);

console.log(document.querySelector(".link-dom").getAttribute("href"));

Ver en la practica de DOM ejemplo.

document.documentElement.lang = "es";

Con esto podemos cambiar el valor de un atributo .

document.documentElement.setAttribute("lang","es-MX");

Este cuando se coloca en la consola cambia el valor del atributo lang. Ya que toma dos párametros el primero el nombre del atributo que se quiera cambiar y el segundo es cual es su nuevo valor.

Es opcional pero cuando le colocamos nombre a nuestra variables "$" usan este simbolo para diferenciar el código de nuestro código que se refiere a nuestro html y cual a la lógica de nuestro código.

Como buena practica cada vez que habramos una nueva ventana tenemos que colocar el atributo rel con el valor noopener para evitar un ataque o robo de información osea que no depende de la pagina nueva que queramos para funcionar.

const $linkDOM = document.querySelector(".link-dom");


$linkDOM.setAttribute("target","_blank");

$linkDOM.setAttribute("rel","noopener");

$linkDOM.setAttribute("href","https://google.com/";);

.hasAttribute("atributo"); con este nosotros sabremos si tiene o no un atributo que le coloquemos en las comillas devolviendo un booleano.

.removeAttribute("rel"); Remueve un atributo que le hayamos colocado.

console.log($linkDOM.getAttribute("data-description"));

console.log($linkDOM.dataset);

console.log($linkDOM.dataset.description);

$linkDOM.setAttribute("data-description","Modelo de Objecto del Documento");

Con estos cuatro métodos podemos ver a los data-attributes  y podes cambiarlo

Ademas se puede eliminar con .removeAttribute y comprobar si ese atributo esta con .hasAttribute().
