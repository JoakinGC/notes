Modificación de Elementos:

.replaceChild() lo que hace es remplazar un nodo por uno nuevo el primer parametro por el nuevo y el segundo por el parametro que se quiere remplazar.

$cards.replaceChild($newCard,$cards.children[2]); // Para remplazar un elemento ya creado

.insertBefore() los parametros es lo mismo que lo anterior pero lo que hace es colocarlo delante del elemento que queremos adelantar.

$cards.insertBefore($newCard, $cards.firstElementChild); // Para colocar adelante de un elemento ya existente.

.cloneNode() tambien como en el templates se puede clonar los elementos pero se debe colocar un true o false o nada para que se comporte de diferentes maneras para que clone todo se debe colocar true. Para que clone solo la etiqueta contenedora solo los parentesis.
