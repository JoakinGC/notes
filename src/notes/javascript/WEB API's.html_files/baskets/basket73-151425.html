Estilos y Variables CSS.

.style si lo colocamos con una variable que guarde la etiqueta que contega el atributo style te va a mostrar todas las propiedades que esten en esta. Aunque tambien se puede hacer llendo con el getAttribute("style") tambien se puede ver todas las propiedades de style. O podemos consultar una a una con .style. Y nombre de la propiedad que queramos saber. .style.color.

Las propiedades CSS se escriben en Camel Case y no con el guion. Ejemplo:

.style.width = "50%";

window.getComputedStyle sirve para poder acceder a las propiedades tambien de css pero tambien se puede acceder a  otro archivos externos. Lo muestra en la consola

Aunque no es necesario colocar window ademas podemos cambiar el valor de la propiedad con .getPropertyValue().

En la herramientas de desarrollador podes ver en computed los valores de las propiedades de CSS en la página ademas de tener

.style.setProperty() le agregamos una propiedad o ya existente y le cambiamos el valor

Se puede guardar valore de  otras propiedades dentro de una variable y luego usarla para cambiar el valor de una propiedad o repetirla

Si queremos consultar propiedades establecidas desde la hoja de estilos externa debemos usar getComputedStyle
