Todas las etiquetas hijos o que esten dentro de una etiqueta padre

.children mostrara todas las etiquetas que estan dentro de la etiqueta padre y colocando entre [] y el numero de la posicion podemos saber cual es el hijo en esa posicion. Tambien esta childNodes que evalua los nodos hijos si se trabaja con nodos.

Ademas tambien podemos saber el elemento padre o la etiqueta  colocando .parentsElement o .parentsNode. Y la diferencia erradica en que en nuestro te muestra un tipo de elemento y otro un nodo.

.firstElement te muestra el primer elemento que este al lado de la etiqueta literalmente.

Miestra que el .firstElementChild te muestra el primer elemento hijo de la etiqueta padre.

.lastElement devuleve el ultimo elemento pero solo lo que contiene y no la etiqueta

.lastElementChild devuelve el ultimo etiqueta hijo.

console.log($cards.previousSibling);

console.log($cards.previousElementSibling);

el primero te muestra lo que hay entre el espacio de la etiqueta a la otra y la .previusElementSibling muestra la etiqueta que esta antes de la nuestra

console.log($cards.nextSibling);

console.log($cards.nextElementSibling);

Al igual que las .previousSibling solo que devuelve el espacio que hay entre la etiqueta y la que sigue o el espacio.
