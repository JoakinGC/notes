Propiedades y Eventos:

Son todas las propiedades que se relacionan con la ventana

console.log(window.innerWidth);

console.log(window.innerHeigh);

console.log(window.innerWidth);

Devuelver el ancho de la ventana de nuestro documento que es visible


console.log(window.innerHeight);


Devuelve el alto de nuestra ventana que es visible.

outerWidth() devuelve el ancho pero de la ventana completa

outerHeight() devuelve el alto de la ventana completa

Todo esto es el evento que sucede cada vez que movemos la ventana

De la misma aplica para el Scroll

.scrollX haciendo referencia al eje x que es el ancho

.scrollY hacinedo referencia al eje Y que es la altura.

Pero existe una alternativa a ese evento llamada "DOMContentLoaded" este hace lo mismo que al anterior e incluso sus métodos son iguales la diferencia erradica en que carga más rápido por lo menos 6 o 7 veces. Por ende a la hora de usar algo que requiera que recarge la página es mejor usar este evento. Más con asincronia.

.screenY el eje que dice la altura que se encuentra la ventana de nuestro navegador de nuestra computadora.

Tambien para evento load o cada vez que recargamos la pagina

.screenX toma como referencia la pantalla principal de la computadora y de ahi si esta en la derecha es positivo si se tienen una segunda pantalla y si se tiene una a la izquierda es negativo siempre diciendo la ubicación del navegador que muestra nuestra posicion
