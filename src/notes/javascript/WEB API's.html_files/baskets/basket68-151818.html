DOM

Document Object Model

Es el objecto encargado de JS de controlar la parte del objecto Documetn o documento de la misma página

No es necesario colocar windows cada vez que invoquemos document

document.head para ver la cabeza

document.body para ver el cuerpo del documento

document.documentElement para todo el cuerpo de html pero no vemos el <!DOCTYPE>

document.doctype para ver el tipo de Doctype que es el archivo

document.charset para ver el tipo de teclado que se usa

document.title para ver el titulo de nuestra página.

document.links para ver los enlaces de nuestra página. No confundir su propiedad length de un array, es un nodo.

document.images para ver todas las imagenes de nuestra página

document.forms te devuelve todos los formularios.

document.styleSheets para ver los estilos de la página.

document.scripts evalua todas las etiquetas scripts que hay en el documento.

document.getSelection te muestra la parte selecionada con el mouse en negro  si se usa en combinacion con setTimeout() y el toString() se puede ver en la console como un string lo que selecionamos.

document.write() para escribir directamente en nuestra pagina pero no buena practica usarla. SIempre que se coloque codigo dentro de este lo coloca al final de todas las etiquetas salvo que sean etiquetas dinamicas.
