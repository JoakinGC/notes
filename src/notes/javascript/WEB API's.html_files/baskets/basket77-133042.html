Creación de Elementos y Fragmentos:

document.createElement() creamos un elemento HTML o una etiqueta y la podemos guardar en una variable.

document.createTextNode() agrega un nodo de texto en la variable que guardemos.

const $figure = document.createElement("figure"),

$img = document.createElement("img"),

$figcaption = document.createElement("figcaption"),

$figcaptionText = document.createTextNode("Animals"),

$cards = document.querySelector(".cards");


//Agregar al Arbol de HTML:


$img.setAttribute("src","https://placeimg.com/200/200/animals";);

$img.setAttribute("alt","animals");

$figure.classList.add("card");


$figcaption.appendChild($figcaptionText);

$figure.appendChild($img);

$figure.appendChild($figcaption);

$cards.appendChild($figure);

guarda un nodo y cada nueva etiqueta en una variable y selecciona una parte del html que tiene una clase llamada cards y luego agregar los atributos src con el link y alt. Figure le agrega una clase card. Y luego lo agrega con appendChil();

const $figure2 = document.createElement("figure");


$cards.appendChild($figure2);


$figure2.innerHTML = `

   <img src="https://placeimg.com/200/200/tech"; alt="Tech">

   <figcaption>Tech</figcaption>

`;


$figure2.classList.add("card");

De esta forma creamos una nuevo elemento figure que agregamos como etiqueta hijo. Y luego le agregamos las otras dos etiquetas con .innerTHML y al final una clases llamada card.

.appendChild() agrega un elmento como una etiqueta hijo.

La ventaja uno con el otro es que en el primero si generamos el nodo de texto y html y entonces podemos interactuar con ellos y con el .innerHTML no es solo visual. Ya debende del proposito que lo usemos.

//Cracion de lista:


const estaciones = ["Primavera","Varano","Otoño","Invierno"],

$ul = document.createElement("ul");

$estaciones = document.createTextNode("Estaciones del Año");

$h3 = document.createElement("h3");



document.body.appendChild($h3);

$h3.appendChild($estaciones);

document.body.appendChild($ul);


estaciones.forEach((el) =>{

   const $li = document.createElement("li");

   $li.textContent = el;

   $ul.appendChild($li);

})

creacion de una lista usando nodos de texto y de elementos html combinados con .forEach();. Ademas nodo de html de la lista tiene una propiedad llamada .textContent para agregar un nuevo valor a la lista del nodo.

const continentes = ["África","América","Asia","Europa","Oceanía"],

$ul2 = document.createElement("ul");


document.write(`<h3>Continentes del Mundo.</h3>`);

document.body.appendChild($ul2);


$ul2.innerHTML = "";

continentes.forEach((el) => ($ul2.innerHTML += `<li>${el}</li>`));

Lo mismo que lo anterior pero diferente el código utilizando .innertHTML. Ademas la lista dos tiene que ya tener algo definido antes de usar el .innertHTML porque sino no sumara nada cuando usamos el +=;

El problema del uso de estas tecnicas el gasto de recursos ya que colocamos en el DOM cada elemento pero si son 100 elementos tendriamos que colocar 100 elementos al DOM y eso es un problema de rendimiento por eso con los fragmentos colocamos los 100 elementos primero y luego colocamos todo solo una vez al DOM.

const meses = [

   "Enero",

   "Febrero",

   "Marzo",

   "Abril",

   "Mayo",

   "Junio",

   "Julio",

   "Agosto",

   "Septiembre",

   "Octubre",

   "Noviembre",

   "Diciembre"

],

$ul3 = document.createElement("ul"),

$frament = document.createDocumentFragment();


meses.forEach((el) =>{

   const $li = document.createElement("li");

   $li.textContent = el;

   $frament.appendChild($li);

})



document.write("<h3>Meses del Año</h3>");

$ul3.appendChild($frament);

document.body.appendChild($ul3);

Esta es la forma correcta ya que es usando fragmentos para evitar consumir tantos recursos para agregar elementos. Se guarda primero en el fragmento y luego lo pasamos como hijo a la lista y luego al body.
