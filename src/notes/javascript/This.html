Dentro del bloque que hay es un nuevo contexto en JS.

Con las funciones declaradas fuera de un objecto se puede simplificar la sintaxis y usar de ayuda pero con las funciones flechas estas no se pueden usar porque genera conflicto con este pero eso no quiere decir que no se pueda usar lo no se puede usar de mnaera directa pero si se guarda la funcion en una variable si se puede

Cada nueva funcion cuando usamos una funcion dentro de otra funcion genera otro contexto por lo que se debe tener esto en cuenta a la hora de usar this.

La idea es usar combinada los diferentes contexto que genera cada funcion para lograr cumplir con la funcion.
