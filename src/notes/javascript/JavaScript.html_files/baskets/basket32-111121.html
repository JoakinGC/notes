Comporatamiento de una funcion prototipicas y su herencia

Entonces creamos una funcion constructora base y de ahi heredamos sus propiedades o atributos para que no tengamos que sobreescribir todo de nuevo ademas la mejor manera para generar métodos en las funciones de tipo prototipica es llamando a la funcion protopico() ademas de cuando generamos la nueva funcion constructura debemos usar la funcion super y respetar su nombre cuando creamos la nuevo funcion constructora.

Un nueva funcion hijo puede heredar los métodos de la funcion padre, y si se quiere generar una nueva funcion es igual que con una funcions constructoria

La funcion hijo puede sobrescribir los métodos ya creados ademas de crear métodos que solo estan en la funcion hijo

//Sobreescritura de métodos de prototipo Padre en hijo.


Perro.prototype.sonar = function(){

               console.log("Soy un perro y mi sonido es un ladrido");

}

// Creacion de una nueva funcion dentro de la funcion constructora de Perro:


Perro.prototype.ladrar = function(){

               console.log("Guauu guauu !!");

}

//Pruebas:


const pepe = new Perro("Pepe","Masculino", "Mediano");
