¿Qué es un array?

Un array o arreglo no es mas que una coleccion de elementos  de diferentes tipos.

¿Desde que posicion incia el primer elemento agragado a un array?

Cada elemento que tiene el arreglo siempre empieza en 0.

¿Qué tipos de datos puede contener un array?

Un array puede contener varios tipos de datos hasta mismo arrays dentro de otros array hasta objectos.

Metodo para contar lo elementos de un array

El metodo .lenght cuenta cuantos elementos tiene el array dentro del mismo.

Formas de busqueda de elementos dentro de un array?

Para buscar un elemento especifico se puede hacer de varia maneras una es si sabemos la posicion del elementos que buscamos que la sintaxis seria ej:

"NombreDelArray.(3);"

El 3 seria la posicion del elemento que buscamos.

¿Que sucede dentro de una array que tiene otro array?

Se pueden buscar array dentro de otro array y asi sucesivamente ver en la practica para entender mejor pero es tan simple como se dice.

Otra forma de crear un array es con el método Array():

const c = Array.of("Elementos que quiero cargar");

Con el podemos usar el fill para decirle con que llenar el array y decirle cuantos elementos queremos:

const c = Array(100).fill(false);


genera cien elementos con false

Tambien otra forma de crear un array es usar su contructor new Array pero ya se dejo esta forma por la fomra literal.

Forma para sumar elementos al array?

Con el método .push () si se usa en un array agrega un nuevo elemento al array

Metodo para sacar elementos de un array?

El método .pop() lo que hace es quitar el ultimo elemento del array

Metodo para visualizar o recorrer los elemento de un array

El método .forEach() muestra los elementos dentro del array y los devuelve si lo combinamos con una funcion flecha o funcion normal que gestione la manera que se muestra los elementos revisar la practica en la parte de Arrays.

Método para mapear los elementos de un array?

Método Map()

Es un método que busca o mapea dentro de un array y te devuelve un valor que nosotros queramos.

let aguas = [{nombre: "Villavicencia", mililitros: 500},

{nombre: "Soda", mililitros: 700},

{nombre: "Aqua", mililitros: 1000}];


let aguasConsumidas = aguas.map(elements => {

             elements.mililitros -=100;

             return elements;

});

console.log(aguasConsumidas);

Es importante que cuando llamamos a map colocar el valor de manera decrementa ya que hace igual al valor porque sino returna un array con la cantidad restante y no mostrara lo nombres y demas propiedades.

De esta forma devulve en forma de objecto con la cantidad de mililitros que se restaron

let aguas = [{nombre: "Villavicencia", mililitros: 500},

{nombre: "Soda", mililitros: 700},

{nombre: "Aqua", mililitros: 1000}];


let aguasConsumidas = aguas.map(elements => elements.mililitros - 100;);

console.log(aguasConsumidas);

¿Método para o filtrar un elemenento de un array?

De manera provisoria explicacion del método map(), filter().

El métod filter es un método que se usa en la programacion funcional en lo que no piensas como funciona sino lo que quieres obtener. Es decir el método que es lo que quieres que devuelva.

let aguas = [{nombre: "Villavicencia", mililitros: 500},

{nombre: "Soda", mililitros: 700},

{nombre: "Aqua", mililitros: 1000}];


let aguasGrandes = agua.filter( elementos => elementos.mililitros > 500);

console.log(aguasGrandes);
