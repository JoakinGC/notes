JSON

JavaScrits Object Notation

JSON es un formato ligero de cambio de datos. Ahora es un estandar

JSON es un formato que se utiliza como medio para que cualquier lenguaje pueda interpretar diferentes datos en comunes

Todos los lenguajes tienen que bases con JSON.

La terminacion JSON es .json los archivos.

Todas las propiedades del objecto van con comillas ("") y tambien los valores menos "null".

.parse("{}") transforma código JSON a JS colocando dentro de la comillas.

JSON.stringify({}) Transforma un objecto literal sin comillas o cualquier otor dato de JS a código de JSON.
