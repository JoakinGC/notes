Comandos y utilidades

Configuración inicial

  • aws configure
    Configura las credenciales de AWS (Access Key ID, Secret Access Key, región y formato de salida).

S3 (Simple Storage Service)

  • aws s3 ls
    Lista los buckets de S3 en tu cuenta.
  • aws s3 mb s3://nombre-del-bucket
    Crea un nuevo bucket.
  • aws s3 cp archivo.txt s3://mi-bucket/
    Sube un archivo a un bucket.
  • aws s3 sync . s3://mi-bucket/
    Sincroniza un directorio local con un bucket.
  • aws s3 rm s3://mi-bucket/archivo.txt
    Elimina un archivo de un bucket.

EC2 (Instancias)

  • aws ec2 describe-instances
    Muestra información sobre las instancias EC2.
  • aws ec2 start-instances --instance-ids i-1234567890abcdef0
    Inicia una instancia específica.
  • aws ec2 stop-instances --instance-ids i-1234567890abcdef0
    Detiene una instancia.

IAM (Identidad y Acceso)

  • aws iam list-users
    Lista todos los usuarios IAM.
  • aws iam create-user --user-name nombreUsuario
    Crea un nuevo usuario.
  • aws iam attach-user-policy --user-name nombreUsuario --policy-arn arn:aws:iam::aws:policy/NombrePolitica
    Asigna una política a un usuario.

CloudFormation

  • aws cloudformation create-stack --stack-name nombreStack --template-body file://plantilla.yaml
    Crea un stack desde una plantilla local.
  • aws cloudformation describe-stacks
    Muestra los detalles de los stacks.

SNS (Notificaciones)

  • aws sns create-topic --name nombreTopic
    Crea un nuevo tema de notificación.
  • aws sns publish --topic-arn arn:aws:sns:region:account-id:nombreTopic --message "Hola"
    Envía un mensaje al tema.

CloudWatch

  • aws cloudwatch list-metrics
    Lista las métricas disponibles.
  • aws cloudwatch get-metric-statistics ...
    Obtiene estadísticas de una métrica específica (requiere varios parámetros).
