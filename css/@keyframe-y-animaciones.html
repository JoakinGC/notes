@keyframe y Animaciones

Regla de Css que nosotros definimos en un @keyframe y cuanta con un nombre, que podemos usar en conjunto con otras propiedades. Para hacer una animación

/* The animation code */  
@keyframes example {  
  from {background-color: red;}  
  to {background-color: yellow;}
 }  
  
/* The element to apply the animation to */  
div {  
  width: 100px;  
  height: 100px;  
  background-color: red;  
  animation-name: example;  
  animation-duration: 4s;
  }

Llama a div y la animación se ejecuta con el div, con duration controlamos el tiempo del mismo. Dentro de las reglas @keyframe podemos ser mas específicos usando porcentaje:

/* The animation code */  
@keyframes example {  0%   {background-color: red;}  
  25%  {background-color: yellow;}  
  50%  {background-color: blue;}  
  100% {background-color: green;}}  
  
/* The element to apply the animation to */  
div {  
  width: 100px;  
  height: 100px;  
  background-color: red;  
  animation-name: example;  
  animation-duration: 4s;
  }

Propiedades de Animaciones

Contamos con animation-delay que define el diley en la animación.

div {  
  width: 100px;  
  height: 100px;  
  position: relative;  
  background-color: red;  
  animation-name: example;  
  animation-duration: 4s;  
  animation-delay: 2s; /*si usamos negativos empezara desde el tiempo que coencida (-2 segundo empezara por la mitad)*/
 }

La animation-iteration-count es la propiedad que permite controla la cantidad de iteraciones o veces que se ejecute la animación.

Podemos definir un numero o hacerlo de manera infinita con el valor infinite

Ejecutar la animación en diferentes direcciones.

Tenemos una serie de propiedades que permiten controlar el orden, la propiedad que hace ello es animation-direction, sus valores son, normal,reverse ejecuta en el orden contrario,alternate primero hacia adelante y luego hacia atrás,alternate-reverse se reproduce primero hacia atrás, y luego hacia adelante.

Controlar la curva de velocidad de la animación.

La propiedad se llama animation-timing-funtion, tiene los siguientes valores:

  • ease: Empieza lento, rápido para terminar lento.
  • linear: Misma velocidad.
  • ease-in: Inicio lento.
  • ease-out : Final lento
  • cubic-bezier(n,n,n,n): Nosotros definimos la velocidad.
